# ICCM 2026

9th International Conference on Contemporary Management (ICCM-2026)

## Overview

This is the official website for ICCM 2026, built with modern web technologies to provide a fast, responsive, and accessible user experience.

## Features

- **Modern Tech Stack**: React 19 + Vite + Tailwind CSS v4
- **Responsive Design**: Mobile-first approach with optimized layouts for all devices
- **Light/Dark Mode**: Toggle between light and dark themes with persistent preferences
- **SEO Optimized**: Meta tags, Open Graph, and semantic HTML for better discoverability
- **Accessible**: ARIA labels, keyboard navigation, and semantic structure
- **Fast**: Optimized build with code splitting and lazy loading

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm run lint
```

## Project Structure

```
src/
├── components/       # Reusable components (Navigation, Footer)
├── pages/           # Page sections (Home, About, etc.)
├── contexts/        # React contexts (Theme)
├── hooks/           # Custom hooks (useTheme)
├── assets/          # Static assets
├── App.jsx          # Main app component
├── main.jsx         # Entry point
└── index.css        # Global styles
```

## Sections

The website includes the following sections:

1. **Home** - Hero section with conference overview
2. **About** - Conference description and highlights
3. **Theme** - Conference theme and focus
4. **Call for Papers** - Submission guidelines
5. **Important Dates** - Key deadlines and dates
6. **Keynote Speakers** - Featured speakers
7. **Committees** - Organizing, Program, and Advisory committees
8. **Focus Areas** - Research topics and themes
9. **Registration** - Fee structure and registration info
10. **Programme** - Conference schedule
11. **Venue** - Location and accommodation details
12. **Sponsors** - Sponsorship opportunities
13. **Contact** - Contact form and information

## Technologies

- **React 19.2** - UI library
- **Vite 7.3** - Build tool and dev server
- **Tailwind CSS 4.1** - Utility-first CSS framework
- **ESLint** - Code linting and formatting
